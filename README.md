#  Guess the Number Game

A simple **Guess the Number** game built with **HTML, CSS, and JavaScript**.  
This mini project is designed to practice basic JavaScript logic, DOM manipulation, and user interaction.

---

##  How the Game Works

- The game randomly generates a number between **1 and 100**
- The player tries to guess the number
- Each guess will give feedback:
  -  Too low
  -  Too high
  -  Correct
- The number of attempts is limited
- The game can be reset at any time

---

##  Features

- Random number generation
- User input validation
- Attempt counter
- Keyboard support (Enter & shortcut keys)
- Responsive and simple UI
- Reset game functionality

---

##  Built With

- **HTML** – structure
- **CSS** – styling & responsiveness
- **JavaScript (Vanilla)** – game logic & DOM manipulation
